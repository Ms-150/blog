## JSBarcode 条形码生成器

JSBarcode 是一个用于生成条形码的 JavaScript 库，支持多种条形码格式.
如 Code 128、EAN、UPC 等。通过简单的配置，你可以将条形码直接渲染到网页中。

### 安装

首先，你需要通过 npm 或 CDN 引入 JSBarcode。

#### 使用 npm 安装

```bash
npm install jsbarcode
```
